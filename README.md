# Get Started

1. In your terminal, run the following commands one-by-one:

```bash
git clone git@github.com:WorkPad-HQ/website.git
cd website
npm install
```

2. Rename `.env.example` to `.env.local`:

```bash
cp .env.example .env.local
```

3. Run the development server:

```bash
npm run dev
```
